su di me

Ho svolto l'Academy Java Developer nel 2023 presso QiBit School.

JAVA

Java è uno dei linguaggi di programmazione della quale possiedo maggiori competenze. Oltre allo studio effettuato durante l'Università, ho appreso questo linguaggio, in maggiore dettaglio, presso un Academy per Java Developer tenutasi tra Febbraio e Marzo del 2023, tenuta dai docenti Mirko Campari e Vincenzo Bruno. In Java, oltre alle nozioni base, possiedo una discreta conoscenza di JDBC, che ho utilizzato per collegare Java e MySQL.

C

C è uno dei linguaggi di programmazione dalla quale sono partito. Ho sviluppato alte conoscenze del linguaggio base presso l'Università. Ho svolto diversi progetti in C sempre durante l'Università e mi è stato molto utile conoscerlo, insieme a Java, per poter approfondire le mie conoscenze di C++.

C++

C++ è un linguaggio molto interessante che sto attualmente studiando. La comprensione di questo linguaggio non mi è molto complessa. Infatti conoscendo già il linguaggio Java e il linguaggio C, non ho particolari difficoltà nell'apprendimento di questo linguaggio. In questo momento non possiedo un'alta padronanza di questo linguaggio, ma riuscirò ad acquisirla in breve tempo.

MySQL

MySQL è uno dei programmi appresi durante l'Academy Java Developer, che ho utilizzato, per l'apprendimento del linguaggio SQL e per l'utilizzo dei database creati e delle query tramite JDBC, atto a collegare questi database a un programma Java.

Unreal engine

Unreal Engine si può dire che è il mio motore grafico preferito, potente e veloce, ma molto complesso. Conosco una parte di questo motore grafico, so muovermi a mio agio durante il suo utilizzo, e voglio continuare a studiarlo per poterlo padroneggiare a pieno, grazie anche allo studio che sto svolgendo nel linguaggio C++.

BLENDER

Blender è la suite di modellazione 3D che conosco, in maniera non completa, ma che prevedo di apprendere a pieno. Utilizzo spesso questo software per la creazione di modelli 3D da utilizzare all'interno del software Unreal Engine 5.

HTML 5

Conosco il linguaggio di markup HTML 5 da anni, e continuo tutt'ora a studiarlo e ad utilizzarlo. Tramite questo linguaggio ho creato il front end del progetto finale svolto durante l'Academy per Java Developer. Trovate il progetto sul mio profilo GitHub.

CSS 3

Ho imparato e continuo a utilizzare il linguaggio di stile CSS 3. Sono abbastanza capace e mi trovo a mio agio nell'utilizzo di questo linguaggio. Durante l'Academy per Java Developer ho utilizzato il linguaggio CSS, unito naturalmente al linguaggio HTML, per la realizzazione del progetto finale che trovate sul mio profilo GitHub.

BOOTSTRAP STUDIO

Utilizzo spesso questo software per la creazione di pagine web statiche e dinamiche. Mi piace come software, in quanto è molto utile nel potenziare le capacità di HTML e CSS e nel velocizzare i processi di creazione delle pagine web. Questo stesso sito sulla quale state navigando è creato in Bootstrap Studio.

GitHub

Github è il software che uso per il versionamento, salvataggio e pubblicazione dei miei progetti. Utilizzo da anni questo software, che permette davvero di tenere una libreria ordinata contenente i miei progetti. Potete cercare il mio profilo GitHub: TheJolly01

CANVA

TRELLO
